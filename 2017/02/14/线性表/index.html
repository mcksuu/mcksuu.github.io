再识数据结构（1）线性表

1. 线性表的定义

线性表（linear list）是数据结构的一种，一个线性表是n个具有相同特性的数据元素的有限序列。数据元素是一个抽象的符号，其具体含义在不同的情况下一般不同。

在稍复杂的线性表中，一个数据元素可由多个数据项（item）组成，此种情况下常把数据元素成为记录（record），含有大量记录的线性表又称文件（file）。

线性表中的个数n定义为线性表的长度，n=0时称为空表。在非空表中每个数据元素都有一个确定的位置，如用ai表示数据元素，则i称为数据元素ai在线性表中的位序。

线性表的相邻元素之间存在着序偶关系。如用$(a1,…,a{i-1},ai,a{i+1},…,an)$表示一个顺序表，则表中$a{i-1}$领先于$a_i$，$ai$领先于$a{i+1}$，称$a_{i-1}$是$ai$的直接前驱元素，$a{i+1}$是$a_i$的直接后继元素。当$i=1,2,…,n-1时，$a_i$有且仅有一个直接后继，当$i=2,3,…,n$时，$a_i$有且仅有一个直接前驱。

{
if (position <= 0 || position > ls->length)
{
printf(“请输入正确的插入位置”);
}
else
{
if (ls->length % DEFAULT_SIZE == 0)//如果当前list的长度和默认的长度相等，需要重新分配内存
{
//重新分配内存，增量为DEFAULT_SIZE
realloc(ls->elem, DEFAULT_SIZE (ls->length / DEFAULT_SIZE + 1) * sizeof(ls->elem));
}

    for (int *i = &(ls->elem[ls->length - 1]); i >= &(ls->elem[position - 1]); i--)
    {
        *(i + 1) = *i;//向后移
    }

    ls->elem[position - 1] = value;//插入位置重新赋值

    ls->length++;//长度加1

}

}

void DeleteElem(ListStruct ls, int position, int value)
{
if (position <= 0 || position > ls->length)
{
printf(“请输入正确的插入位置”);
}
else
{
value = ls->elem[position - 1];//删除的元素的值
int p = &ls->elem[position - 1];
int *q = &ls->elem[ls->length - 1];

    while (p < q) {
        *p = *p + 1;//向前移动
        p++;
    }

    ls->length--;//长度-1
}

}

void DestoryList(ListStruct * ls)
{
free(ls->elem);//释放线性表，这里不能去释放ls，这个内存并不是我们申请的
